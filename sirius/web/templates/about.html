About

This is an alternate Little Printer backend, maintained by Nord Projects.

You can:

  • Claim your printer
  • Send test messages
  • Use it with our Little Printers app to send messages from your iPhone!
  • Send messages via our Print Key API

To use your Little Printer with this backend, you'll need to root your bridge, and install a software update. Check out information about the process or check out a step-by-step guide here.

All the code is open source, check it on GitHub. This backend is forked from the sirius project, and most of the code originated there. Thanks to Matt Webb, Tom Hunger, Quentin Dufour, who did the lion's share of the work to get this thing working!

Finally, thanks to the good people of BERG, whose product captured our imaginations.
